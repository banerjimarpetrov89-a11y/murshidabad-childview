# ChildWatch AI

AI-Powered Child Protection Intelligence & Situational Analysis Platform

## Live Demo
https://murshidabad-childview.lovable.app

## Problem

Child protection data is often fragmented across multiple systems, making it difficult to identify vulnerability patterns, emerging concerns and geographic hotspots.

## Solution

ChildWatch AI transforms district-level child protection indicators into intelligence through:

- Vulnerability Analysis
- Hotspot Mapping
- AI-generated Situational Signals
- Child Protection Intelligence Assistant
- District Intelligence Brief Generator
- Risk Signal Detection

## Technology Stack

- Lovable
- React
- TypeScript
- Supabase
- AI-powered analytics

## Privacy

The platform uses aggregated, non-personal indicators and does not process personally identifiable information.
